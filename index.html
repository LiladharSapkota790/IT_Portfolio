Projects & Case Studies

Setting up Network & Domain Configuration

Problem: A company required multiple devices to be connected to the network and domain with user access configured.

Solution:

  • Configured network devices, including routers and switches, ensuring proper connectivity.
  • Installed Windows on client machines and joined them to the domain.
  • Created user accounts in Active Directory and assigned necessary permissions.
  • Configured Office365 accounts and assigned licenses to users.
  • Tested user authentication to ensure proper login and domain connectivity.

Outcome: The network was successfully deployed, allowing seamless user access and efficient management.

Laptop Battery Replacement & Troubleshooting

Problem: The gaming laptop was shutting down unexpectedly despite showing a high charge.

Solution:

  • Used battery diagnostic tools to assess battery health.
  • Identified a degraded battery and searched for a compatible replacement.
  • Safely removed the old battery, ensuring proper disconnection procedures.
  • Installed the new battery and performed a full charge-discharge cycle.
  • Calibrated battery settings in BIOS and monitored performance over time.

Outcome: The laptop now runs efficiently with improved battery life.

POS System Troubleshooting

Problem: POS system not processing transactions and frequently crashing.

Solution:

  • Checked for software corruption and updated the POS system.
  • Verified network connectivity and reconfigured settings.
  • Reinstalled and optimized POS software to reduce crashes.
  • Provided training to staff on proper usage and quick troubleshooting.

Outcome: The system was restored to full functionality, reducing downtime.

Active Directory & Office365 User Management

Problem: A new employee was unable to log in, and existing users had email sync issues.

Solution:

  • Created a new user account in Active Directory and assigned proper permissions.
  • Configured mailbox settings and linked them to Office365.
  • Checked authentication settings and resolved credential-related issues.
  • Enabled security policies and guided the user through the setup process.

Outcome: Successful employee onboarding and restored Office365 functionality.
